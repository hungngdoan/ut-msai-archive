## The idea

The fastest way for me to stop treating attention as magic is to implement it from nothing. This is a
small, deliberately un-optimized transformer: the point is clarity, not speed or benchmarks. It is a
personal study project built before the term, not coursework.

## What's inside

- A from-scratch scaled dot-product attention with explicit shape checks
- Multi-head attention, positional encodings, and a minimal training loop
- A character-level language model trained on a small, public-domain text dataset

## Status and honesty

It trains and produces plausibly structured text at small scale; it is not competitive with anything
real, and that's fine. The deliverable is understanding. Next I want to add a clean evaluation script
and a short write-up comparing my hand-rolled attention against the framework implementation to make
sure they agree numerically.
